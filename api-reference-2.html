Endpoint

Filtrer les missions

Endpoint

GET /api/missions/:id

Description

Récupère les missions d'un utilisateur en fonction de leur date : passées, en cours ou futures. Possibilité de limiter les résultats.

Authorization

La méthode d’authentification est requise pour cette requête API.

Type

Required?

Description

Exemple de réponse

{ "past": [ ... ], "current": [ ... ], "future": [ ... ] }
