Référence de l'API de Pointage

Cette API permet de gérer les sessions de travail des employés, qu'elles soient en temps réel ou déclarées manuellement. Elle couvre les actions suivantes :

  • Démarrer une session de travail

  • Mettre une session en pause

  • Reprendre une session

  • Arrêter une session

  • Créer une session manuelle

  • Obtenir les sessions par mission

  • Obtenir les sessions sans mission

Authentification

⚠️ Toutes les requêtes doivent être authentifiées à l'aide d'un token JWT dans l'en-tête Authorization.

Authorization: Bearer <votre_token>
